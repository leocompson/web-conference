Click on the login to start or join a meeting

This app uses a Websocket server for signaling purposes (connecting peers via WebRTC). The default Websocket server is from - wss://connect.meetingserver.repl.co. If you leave the token area empty, the app will use the default value.

You can change the server to a desired one at any time. Please enter a valid server address and related token in the above input fields. All changes are auto-saved to the memory.
